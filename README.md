Bank Account Management System 🏦
A robust C# Console Application designed to manage banking operations efficiently. This project demonstrates the implementation of Object-Oriented Programming (OOP), Event-driven architecture, and JSON serialization in .NET.

✨ Features
Account Management:

Create unique bank accounts with ID, Name, and Email validation.

Delete existing accounts.

View detailed account information (Balance, Creation Date, etc.).

Financial Operations:

Dispose (Withdraw): Deduct money with balance sufficiency checks.

Transfer: Add or move funds between logic.

Data Persistence: * Automatic data loading on startup.

Save all changes to a BankAccounts.json file upon exiting.

Interactive UI: * Color-coded console output using the Pastel library for better readability.

Event-driven notification system for successful operations.
